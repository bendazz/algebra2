Intercepts of Quadratics

For each quadratic, find the y-intercept (set \( x = 0 \)) and the x-intercepts (set \( y = 0 \) and solve).

Factorable Quadratics

1. Find the x- and y-intercepts of \( y = x^2 - 5x + 6 \).
Reveal Solution

y-intercept: Set \( x = 0 \):

$$ y = 0 - 0 + 6 = 6 $$

So the y-intercept is \( (0, 6) \).

x-intercepts: Set \( y = 0 \) and factor:

$$ x^2 - 5x + 6 = 0 $$ $$ (x - 2)(x - 3) = 0 $$

y-intercept: \( (0, 6) \)   x-intercepts: \( (2, 0) \) and \( (3, 0) \)

2. Find the x- and y-intercepts of \( y = 2x^2 + 7x - 4 \).
Reveal Solution

y-intercept: Set \( x = 0 \):

$$ y = 0 + 0 - 4 = -4 $$

So the y-intercept is \( (0, -4) \).

x-intercepts: Set \( y = 0 \) and factor:

$$ 2x^2 + 7x - 4 = 0 $$

We need two numbers that multiply to \( 2 \times (-4) = -8 \) and add to \( 7 \). Those are \( 8 \) and \( -1 \):

$$ 2x^2 + 8x - x - 4 = 0 $$ $$ 2x(x + 4) - 1(x + 4) = 0 $$ $$ (2x - 1)(x + 4) = 0 $$

y-intercept: \( (0, -4) \)   x-intercepts: \( \left(\dfrac{1}{2},\, 0\right) \) and \( (-4, 0) \)

Irrational Roots (Quadratic Formula)

3. Find the x- and y-intercepts of \( y = x^2 - 4x + 1 \).
Reveal Solution

y-intercept: Set \( x = 0 \):

$$ y = 0 - 0 + 1 = 1 $$

So the y-intercept is \( (0, 1) \).

x-intercepts: Set \( y = 0 \). This doesn't factor nicely, so use the quadratic formula with \( a = 1,\; b = -4,\; c = 1 \):

$$ x = \frac{4 \pm \sqrt{16 - 4}}{2} = \frac{4 \pm \sqrt{12}}{2} = \frac{4 \pm 2\sqrt{3}}{2} = 2 \pm \sqrt{3} $$

y-intercept: \( (0, 1) \)   x-intercepts: \( (2 - \sqrt{3},\, 0) \approx (0.27,\, 0) \) and \( (2 + \sqrt{3},\, 0) \approx (3.73,\, 0) \)

4. Find the x- and y-intercepts of \( y = -3x^2 + 6x + 2 \).
Reveal Solution

y-intercept: Set \( x = 0 \):

$$ y = 0 + 0 + 2 = 2 $$

So the y-intercept is \( (0, 2) \).

x-intercepts: Set \( y = 0 \) and use the quadratic formula with \( a = -3,\; b = 6,\; c = 2 \):

$$ x = \frac{-6 \pm \sqrt{36 + 24}}{-6} = \frac{-6 \pm \sqrt{60}}{-6} = \frac{-6 \pm 2\sqrt{15}}{-6} = \frac{3 \mp \sqrt{15}}{3} $$

Note the sign flip when dividing by \(-6\).

y-intercept: \( (0, 2) \)   x-intercepts: \( \dfrac{3 - \sqrt{15}}{3} \approx -0.29 \) and \( \dfrac{3 + \sqrt{15}}{3} \approx 2.29 \)

Special Cases

5. Find the x- and y-intercepts of \( y = x^2 + 2x + 5 \).
Reveal Solution

y-intercept: Set \( x = 0 \):

$$ y = 0 + 0 + 5 = 5 $$

So the y-intercept is \( (0, 5) \).

x-intercepts: Set \( y = 0 \) and check the discriminant:

$$ b^2 - 4ac = (2)^2 - 4(1)(5) = 4 - 20 = -16 $$

The discriminant is negative, so there are no real x-intercepts. The parabola opens up (\( a = 1 > 0 \)) and sits entirely above the x-axis.

y-intercept: \( (0, 5) \)   x-intercepts: none (discriminant \( < 0 \))

6. Find the x- and y-intercepts of \( y = 4x^2 - 12x + 9 \).
Reveal Solution

y-intercept: Set \( x = 0 \):

$$ y = 0 - 0 + 9 = 9 $$

So the y-intercept is \( (0, 9) \).

x-intercepts: Set \( y = 0 \). Recognize this as a perfect square:

$$ 4x^2 - 12x + 9 = (2x - 3)^2 = 0 $$ $$ 2x - 3 = 0 \implies x = \frac{3}{2} $$

There is exactly one x-intercept (a repeated root). The vertex of the parabola sits on the x-axis.

y-intercept: \( (0, 9) \)   x-intercept: \( \left(\dfrac{3}{2},\, 0\right) \) (double root)
