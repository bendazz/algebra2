Quadratic Parent

1. Sketch the parent function and state the transformations: \( g(x) = (x - 4)^2 + 3 \)
Reveal Solution

Parent function: \( f(x) = x^2 \)

Transformations:

  • Horizontal shift right 4 units
  • Vertical shift up 3 units
2. Sketch the parent function and state the transformations: \( g(x) = -(x - 2)^2 + 5 \)
Reveal Solution

Parent function: \( f(x) = x^2 \)

Transformations:

  • Reflection over the x-axis
  • Horizontal shift right 2 units
  • Vertical shift up 5 units

Square Root Parent

3. Sketch the parent function and state the transformations: \( g(x) = \sqrt{x - 1} + 2 \)
Reveal Solution

Parent function: \( f(x) = \sqrt{x} \)

Transformations:

  • Horizontal shift right 1 unit
  • Vertical shift up 2 units

The new starting point moves from \( (0,0) \) to \( (1,2) \).

Transformations:

The new starting point moves from \( (0,0) \) to \( (-3,1) \).
